1. **Clone o repositório**  
   ```bash
   git clone https://github.com/lauraandradev/api-dev.git
   cd api-dev

2. **Instale as dependências**
   ```bash
   npm i

4. **Compile o TypeScript**
   ```bash
   npx tsc

  ## ▶️ Como Rodar
    npm start
